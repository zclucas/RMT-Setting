## ⚙️ RMT 配置下载与导入

1. 在仓库中找到所需的 `.rmt` 配置文件并下载到本地  
2. 运行 **若梦兔（RMT）** 软件  
3. 点击右上角的 **配置管理 → 导入配置**  
4. 选择下载的 `.rmt` 文件即可完成导入  

---

## ☁️ RMT 配置共享与上传

### 🧩 方式一（适合普通用户）

1. 在对应配置下完善 **使用说明 & 署名** 文件内容  
2. 使用软件内的 **导出配置** 功能  
3. 将导出的 `.rmt` 文件发送到交流群中  
4. **@群主或管理员** 提交审核  
5. 建议在 **QQ频道 → 若梦兔RMT → 共享仓库板块** 发布相关信息

### 🧰 方式二（适合开发者 / 贡献者）

1. **Fork** 本仓库（RMT-Setting）  
2. 按仓库中的目录结构上传你的配置文件  
3. 提交更新后向主仓库发起 **Pull Request**  
4. 管理员审核通过后将合并入主仓库  
5. 建议在 **QQ频道 → 若梦兔RMT → 共享仓库板块** 发布相关信息
