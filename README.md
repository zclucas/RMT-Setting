## RMT配置下载&导入

1. 在仓库文件中查找自己所需的宏配置，下载.rmt配置文件到本地
2. 运行若梦兔软件，点击右上角配置管理，点击导入配置，选择之前下载的.rmt文件即可

## RMT配置上传

### 若梦兔交流群

1. 找到群管理员说明需要上传共享的配置功能
2. 在管理员的指导下完善配置信息
3. 配置审核完成后，由群管理员上传到仓库

### 开源仓库

1. fork RMT-Setting仓库
2. 按配置上传格式增加自己的配置
3. 提交到自己仓库，然后向RMT-Setting发送pull requset请求
4. 仓库管理员审核通过后合并到RMT-Setting中
